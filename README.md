# ❤️ Heart Disease Prediction using Machine Learning

## Overview

This project predicts the likelihood of heart disease using clinical patient information. Multiple machine learning classification algorithms were evaluated, and the best-performing model (K-Nearest Neighbors) was deployed using Streamlit.

---

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Encoding
- Feature Scaling
- Model Comparison
- Heart Disease Prediction Web App
- Streamlit Deployment

---

## Dataset

The project uses the Heart Disease dataset containing patient health records such as:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise-Induced Angina
- Oldpeak
- ST Slope

Target:
- Heart Disease (No)

## Best Model

**K-Nearest Neighbors (KNN)**

Performance:

- Accuracy: **88.59%**
- F1 Score: **89.86%**
